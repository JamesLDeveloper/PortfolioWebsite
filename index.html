About Me

I've been interested in software from a young age and throughout my teens I spent time modding games to tailor them to my preferences.

Being a Christian and having a desire to help people led me to a career where I spend most of my time interacting with people and supporting their development.

In looking for a career adjustment a few friends and family recommended I invest my time in learning software development.

I have really enjoyed the challenge and have spent my free time learning new skills.

I recently started working with a tech startup where I have gained invaluable real life experience and put my new skills to use!

In an ideal world I'd love to spend the majority of my time software developing but still keep a day or two a week for helping others.

Projects

This is a sample of projects I have created as part of my development courses. Click on the title of each project to look at the code hosted on github.

  • Life-Blog Website:
    This site uses HTML and CSS to display text and images using flexbox, grid and a css checkbox on smaller screens so the user can choose to show or hide additional pictures.
  • Codecademy Piano Project:
    Project to create a piano that guides a player through how to play happy birthday song. This project uses getElementByID and eventListeners to assign different values to elements based on where the user has reached in the song.
  • Medieval Marvels and Might:
    This is an android app project based on java with several screens the user can switch between. This project uses different classes, interfaces and generics. It uses on click listeners to respond to user inputs.
  • Unquote Game Android App:
    This game presents users with a series of pictures with a false quote. Users must choose between four different possible answers for who actually said the quote. It tracks their score as they progress through the questions.
